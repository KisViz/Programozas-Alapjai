[⭐] 15. Tüzijáték (3 pont)

Egy nagy tüzijátékot tartanak, amelyben több rakétát lőnek fel egymás után. A kérdés, hogy meddig tart.
A probléma már meg van oldva, de szeretnénk elrontani. Módosítsd a programot, hogy a fordító az alábbi hibákat jelezze:
warning: return type defaults to 'int'
error: subscripted value is neither array nor pointer nor vector
warning: control reaches end of non-void function
