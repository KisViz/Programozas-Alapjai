[⭐⭐] 20. Szegély (6 pont)

A képeknek általában van egy keretük. Most karakterekből fogunk képet készíteni. A szegely függvény feladata, hogy létrehozzon egy megfelelő méretű 2 dimenziós karaktertömböt, majd feltöltse azt megfelelő értékekkel. A függvény paraméterei:
  • sorok: hány sorból álljon a tömb
  • oszlopok: hány oszlopból álljon a tömb
  • normalBetu: melyik betű szerepeljen a tömb közepén (ami nem a széle)
  • szegelyBetu: melyik betű szerepeljen a tömb szegélyén
A függvény hozza létre a tömböt, majd térjen vissza vele.

Például:
c c c c c c c c c c
c S S S S S S S S c
c S S S S S S S S c
c c c c c c c c c c
A fenti példában egy 4 sorból és 10 oszlopból álló tömböt kellett létrehozni. A közepét az 'S', míg a szegélyét a 'c' betűvel kellett feltölteni.
