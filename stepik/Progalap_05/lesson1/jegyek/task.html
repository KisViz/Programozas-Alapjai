[⭐] 9. Jegyek (2 pont)

Tamás sajnálatos módon nem áll túl jól matematikából. A félév vége előtt szeretné összesíteni a jegyeit.
A jegyek függvény feladata, hogy meghatározza és visszaadja, hogy Tamás jegyeinek hányadrésze 1-es (pl. ha három jegye van és ebből 2 egyes, akkor az eredmény 0.666...).

A program sajnos megfelelően működik. Rontsuk el! Alakítsd át úgy, hogy az alábbi hibákat jelezze a fordító:
warning: return type defaults to 'int'
error: called object 'jegyek' is not a function or function pointer
Vajon miből tudjuk, hogy egy azonosító függvényt, vagy valami mást jelöl?
